# Projeto CI/CD Base

Um projeto base para aprender **Integração Contínua (CI)** e **Entrega Contínua (CD)** utilizando Git, GitHub, GitHub Actions e Vercel.

## Sobre o Projeto

Este repositório foi criado como base para um projeto educacional que aborda:

- Controle de versão com Git
- Trabalho com branches e merge
- Commits semânticos
- GitHub Actions para automação
- Deploy automático com Vercel
- Versionamento semântico (SemVer)
- Gerenciamento de issues no GitHub

## Objetivo

Este projeto serve como **ponto de partida** para os estudantes aprenderem na prática como aplicar conceitos de CI/CD em um projeto real.

## Estrutura do Projeto Base
 
```
projeto-cicd-base/
├── index.html          # Página inicial
├── style.css           # Estilos CSS
├── README.md          # Este arquivo
```

## Tecnologias Utilizadas

- **HTML5** - Estrutura da página
- **CSS3** - Estilização e responsividade
- **Git** - Controle de versão
- **GitHub** - Repositório remoto
- **GitHub Actions** - Automação de workflows
- **Vercel** - Hospedagem e deploy contínuo
